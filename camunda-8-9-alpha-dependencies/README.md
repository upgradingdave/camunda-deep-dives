# Camunda 8.9 sdk dependencies

It's highly recommended to use a dependency manager like Maven or Gradle to manage dependencies, but here's a zip containing all dependencies necessary to create a Camunda 8.9 alpha client. This is provided as a convenience in case you want to quickly test out the new features in 8.9 alpha without having to set up a maven or gradle project.
